

## Installation

This is an [ExpressJS](https://expressjs.com/) project.


  Create Environment:

```bash
$ cp .env.dev .env
```

  Install dependencies:

```bash
$ npm install
```

  Watch the server:

```bash
$ npm run serve
```

  Start the server:

```bash
$ npm run start
```


  Lint project:

```bash
$ npm run lint
```

  Test REST with postman:
  
  >make sure to run the application in development mode

```bash
$ npm run newman
```
